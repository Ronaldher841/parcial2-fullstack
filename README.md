# 📌 Evaluación Parcial II - Aplicaciones Web



## 👨‍💻 Descripción

Este proyecto consiste en una aplicación web full stack desarrollada con:



- Frontend: Angular

- Backend: Node.js con Express

- Base de datos: PostgreSQL



El sistema permite autenticación de usuarios y gestión de perfil.



---



## 🚀 Funcionalidades



### 🔐 Login

- Validación de usuario/correo y contraseña

- Verificación contra base de datos

- Manejo de errores



### 👤 Perfil

- Crear perfil

- Actualizar perfil

- Validaciones:

&#x20; - Campos obligatorios

&#x20; - Correo válido

&#x20; - Edad numérica

&#x20; - Teléfono de 8 dígitos



---



## 🛠️ Tecnologías utilizadas



- Angular

- Express
